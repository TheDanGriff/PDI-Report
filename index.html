About This Report

This comprehensive analysis presents the Pitcher Deception Index (PDI) - the first objective metric that quantifies pitch deception in baseball using machine learning and MLB Statcast data.

Key Findings:

  • PDI achieves 98.5% accuracy in identifying deceptive pitch sequences
  • High-deception pitchers show 6.1 pp higher whiff rates
  • Elite pitchers like deGrom, Bieber, and Cole dominate the deception rankings
  • Deception translates to 5.4 WAR difference between elite and poor performers
